ANSWER B:
**Recommendation**
Use the runtime REST API docs generator plus the canonical `AbstractRestHandler` job-result example as the contract anchor.

The response-body rule is: `handleRequest(...)` returns a `CompletableFuture<P extends ResponseBody>`, and the HTTP status comes from the message headers, not from the handler body. That is explicit in `AbstractRestHandler` and is exercised by the job-result handler pattern.
