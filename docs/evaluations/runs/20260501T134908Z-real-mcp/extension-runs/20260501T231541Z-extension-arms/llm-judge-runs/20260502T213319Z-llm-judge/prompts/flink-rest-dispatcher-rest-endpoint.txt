ANSWER B:
**Recommendation**
Dispatcher REST handlers are added in `DispatcherRestEndpoint.initializeHandlers(...)`, where it takes `super.initializeHandlers(...)` and appends the dispatcher-specific `JobSubmitHandler`. The actual REST endpoint registration happens one layer up in `RestServerEndpoint.start()`, which sorts the handler list, checks uniqueness, and calls `registerHandler(...)` for each entry. Versioned route expansion is handled by `getHandlerRoutes(...)`.
