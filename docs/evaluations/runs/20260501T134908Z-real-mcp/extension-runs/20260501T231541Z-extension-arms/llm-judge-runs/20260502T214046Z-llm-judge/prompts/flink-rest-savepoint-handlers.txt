**Implementation anchors**
- `SavepointHandlers` is the implementation class to cite for the trigger/poll flow. Its class Javadoc describes the two-step async behavior, `POST /jobs/:jobid/savepoints` for trigger, and `GET /jobs/:jobid/savepoints/:savepointtriggerid` for polling.  
  - `repo://apache/flink@release-2.2.0/flink-runtime/src/main/java/org/apache/flink/runtime/rest/handler/job/savepoints/SavepointHandlers.java#L64-L115`
- The trigger path is in `SavepointTriggerHandler`: it validates the target directory, derives `TriggerSavepointMode.SAVEPOINT` or `CANCEL_WITH_SAVEPOINT`, and calls `gateway.triggerSavepoint(..., RpcUtils.INF_TIMEOUT)`.  
  - `repo://apache/flink@release-2.2.0/flink-runtime/src/main/java/org/apache/flink/runtime/rest/handler/job/savepoints/SavepointHandlers.java#L218-L259`
- The poll path is in `SavepointStatusHandler`: it calls `gateway.getTriggeredSavepointStatus(key)` and maps `SUCCESS` to a completed `SavepointInfo`, `FAILURE` to a completed failure payload, `IN_PROGRESS` to in-progress, and `UnknownOperationKeyException` to HTTP 404.  
  - `repo://apache/flink@release-2.2.0/flink-runtime/src/main/java/org/apache/flink/runtime/rest/handler/job/savepoints/SavepointHandlers.java#L262-L347`
- The REST shape is fixed by the headers:
  - Trigger: `SavepointTriggerHeaders` -> `POST /jobs/:jobid/savepoints`, `202 ACCEPTED`.  
    `repo://apache/flink@release-2.2.0/flink-runtime/src/main/java/org/apache/flink/runtime/rest/messages/job/savepoints/SavepointTriggerHeaders.java#L27-L84`
  - Poll: `SavepointStatusHeaders` -> `GET /jobs/:jobid/savepoints/:triggerid`, `200 OK`.  
    `repo://apache/flink@release-2.2.0/flink-runtime/src/main/java/org/apache/flink/runtime/rest/messages/job/savepoints/SavepointStatusHeaders.java#L27-L79`
- `WebMonitorEndpoint` wires the handlers into the REST endpoint, so it is the runtime registration point worth citing alongside the class.  
  - `repo://apache/flink@release-2.2.0/flink-runtime/src/main/java/org/apache/flink/runtime/webmonitor/WebMonitorEndpoint.java#L561-L573`
